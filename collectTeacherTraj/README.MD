# Collect Teacher Trajectories

Hello, researcher. We provide the instructions to generate the teacher trajectories using [AgentLab](http://www.github.com/servicenow/agentlab). You can also entirely avoid this step by using our GPT-5 agent trajectories, which can be downloaded from [huggingface](https://huggingface.co/datasets/sparklabutah/TimeWarp-GPT5-Traces).

⚠️ This is not the only method for collecting trajectories. You can also extract trajectories from the traces generated by AgentLab's generic agent. However, we found the way below to be simpler. ⚠️

## Installation

Install AgentLab in editable mode:

```
git clone https://github.com/servicenow/agentlab
cd agentlab
pip install -e .
```

## Modify AgentLab

AgentLab doesn't have a direct way of saving training trajectories in the format we want (apart from execution traces). To enable trajectory saving, we modify/add these two files:

- Replace the [`AgentLab/src/agentlab/experiments/loop.py`](https://github.com/ServiceNow/AgentLab/blob/cbc35a9bc0facaf731bc858c5825edbe757c719f/src/agentlab/experiments/loop.py) file in AgentLab with the provided[`collectTracherTraj/loop.py`](loop.py).

- Add the [`collectTeacherTraj/generic_agent_with_training.py`](generic_agent_with_training.py) in the [`AgentLab/src/agentlab/agents/generic_agent/`](https://github.com/ServiceNow/AgentLab/tree/cbc35a9bc0facaf731bc858c5825edbe757c719f/src/agentlab/agents/generic_agent) directory.

## Inference

Run benchmarks with the `GenericAgentWithTrainingArgs` instead of `GenericAgent`. Import the class using:

```
from agentlab.agents.generic_agent.generic_agent_with_training import GenericAgentWithTrainingArgs
```
